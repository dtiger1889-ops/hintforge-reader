# hintforge-reader

Spoiler-controlled hint companion for any video game guide you have installed. Runtime skill -- pairs with the [`hintforge`](https://github.com/dtiger1889-ops/hintforge) guide builder.

> README in progress. Full installation, do-this-now block, and verification details land at the end of Phase 2 of the [reader-extraction transition](https://github.com/dtiger1889-ops/hintforge_dev). Until then, install paths live under `docs/install/` per runtime.

## What this is

A Claude Code / OpenClaw / Codex skill that loads a game guide (in Hintforge format) and acts as a graduated-spoiler hint companion at runtime. Two dials (enemy 0-5, puzzle 0-3) control how much you want spoiled.

## What this is not

This skill does not build guides. To build a guide for a game from scratch, install the [`hintforge`](https://github.com/dtiger1889-ops/hintforge) builder skill.
